NOTE TAKING APPLICATION
This app is an android APP which yoi can use to 
save your notes for and reuse later.


* FEATURES
1. MVVM
2. Navigation Component
3. ROOM Database
4. Safe args
5. Swipe to delete
6. Recycler view
